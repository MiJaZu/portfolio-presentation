## 🚀 Installation and Development

```bash
# Clone the repository
git clone https://github.com/MiJaZu/portfolio-presentation.git
cd portfolio-presentation

# Install dependencies
npm install

# Run in development
npm run dev

# Build for production
npm run build

# Preview build
npm run preview
```
